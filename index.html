Git commands

  • touch filename (Создать новый файл в текущей директории)
  • git init (Инициализация git проекта в директории)
  • git config --global user.name 'UserName' (Ввод имени пользователя)
  • git config --global user.email 'UserEmail' (Ввод электронной почты пользователя)
  • git add FILENAME (Добовление файла в INDEX для того, чтобы потом его отправить на удаленный репозиторий)
  • -- git add *.html (Добавление всех файлов с расширением .html в область подготовки файлов к отправлению на удаленный репозиторий)
  • -- git add . (Добавление всех файлов в область отправки на удаленный репозиторий)
  • git status (Показывает какие файлы отслеживаются и какие файлы могли быть изменены или какие фалы нужно добавить чтобы потом закомитить)
  • git rm --cached FILENAME (Удаляет файл из отслеживаемых файлов)
  • git commit (Отправка на удаленный репозиторий)
  • -- git commit -m "Your message here" (Отправка на удаленный репозиторий с комментарием внутри консоли)
